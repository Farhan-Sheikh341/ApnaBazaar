ApnaBazaar
ApnaBazaar is a fully functional e-commerce web application built using Django.
It includes essential features such as user authentication, product management, categories, cart system, and product media handling.
The project is structured with clean code practices, modular Django apps, and an extendable architecture suitable for real-world growth.

Key Features
* User Authentication (Login, Register, Logout)
* Product Management (Listings, Product Detail Page, Basic Variants)
* Shopping Cart System (Add to Cart, Remove, Update Quantity)
* Dynamic Categories
* Wishlist Module
* Media Handling for product images
* Responsive Frontend
* Clean URL structure following Django best practices
* Custom Templates setup
* Reusable Apps (accounts, cart, product, home, categories)

Tech Stack
* Backend: Django
* Frontend: HTML, CSS, JavaScript
* Database: SQLite (development)
* Version Control: Git & GitHub
